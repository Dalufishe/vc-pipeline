# VC pipeline

### 資料溝通標準

#### 格式

```js
{
    // data 用於在 pipeline 中傳遞資料
    "data": {
        // ...傳遞的任何資料, 可以是任何 json 物件結構
    },
    // view 用於在顯示預覽
    "view": {
        // 文字預覽
        "text": "Vesuvius Challenge 2024",
        // 圖片預覽 (指定圖片路徑)
        "image": "D:/vesuvius-project/vc-pipeline/public/ferrari.webp",
    }
}
```
